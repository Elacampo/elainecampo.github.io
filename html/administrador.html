Bienvenido a la Página de Administración

En esta página, como administrador, tienes acceso a varias funcionalidades importantes:

  • Visualizar y gestionar la lista de usuarios registrados en el sistema.
  • Navegar entre las diferentes páginas de usuarios para revisar sus datos.
  • Acceder a información detallada sobre cada usuario, incluyendo su nombre, apellido, nombre de usuario, y rol.

El rol de administrador implica responsabilidades clave como la gestión de usuarios, la supervisión de actividades dentro del sistema, y la garantía de que el sistema funcione de manera eficiente y segura.

Utiliza los botones de navegación para moverte entre las páginas de usuarios y asegúrate de cerrar sesión cuando hayas terminado tus tareas.
